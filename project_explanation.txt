Project Explanation Guide: Hexagon Portfolio

1. Project Structure
-------------------
src/
├── components/
│   ├── HexagonButton.jsx      # Custom moon-shaped navigation component
│   └── pages/
│       ├── Home.jsx           # Landing page with moon navigation
│       ├── About.jsx          # Personal information section
│       ├── Experience.jsx     # Work experience showcase
│       └── Contact.jsx        # Contact form with EmailJS integration
├── styles/
│   └── global.css            # Global styles and animations
├── App.jsx                   # Main application component with routing
└── main.jsx                 # Application entry point
